## 인턴십 프로젝트 
### 프로젝트 설명
***
#### 개요
- 계층형 게시판 만들기
- 게시글에 답글을 생성 할 수 있다.
- 답글에 답글을 생성 할 수 있다.
- 게시글 & 답글에 댓글을 생성 할 수 있다.
#### 요구사항
- 회원가입 & 로그인
- 게시판
  - 목록
    - 검색
    - 페이징
    - 정렬
  - 게시글 조회
    - 댓글 쓰기(비동기)
  - 게시글 생성
    - 새글 생성
    - 답글 생성
  - 게시글 수정
  - 게시글 삭제
#### 개발자
- 황제

- 2025.05.19 ~ 2025.06.04
